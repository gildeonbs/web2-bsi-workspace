
# Web II - BSI Workspace

This repository contains projects developed for the **Web II** course in the **Bachelor of Information Systems (BSI/MIS)** program.

The projects are based on tutorials and guides from the official **ASP.NET Core documentation**, with the goal of learning and practicing different web development approaches in .NET.

---

## 📁 Projects

- **MVC Application**  
  Project based on ASP.NET Core MVC tutorials, demonstrating the Model-View-Controller pattern.

- **Razor Pages Application**  
  Built following Razor Pages guides, focusing on page-based development.

- **Blazor Application**  
  Planned project using Blazor for building interactive web UIs with C#.

---

## 🛠️ Technologies

- C#
- ASP.NET Core
- MVC Pattern
- Razor Pages
- Blazor
